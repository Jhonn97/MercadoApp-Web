MercadoApp

O MercadoApp é a solução definitiva para quem busca economia e organização. Diferente de listas comuns, nosso sistema utiliza inteligência para controlar seu orçamento em tempo real.

Descrição Técnica

Desenvolvido em Kotlin com arquitetura nativa Android, o sistema integra recursos de câmera (OCR) e banco de dados local para garantir performance máxima mesmo sem internet.
